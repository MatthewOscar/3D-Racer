# Apex Drift 3D

A browser-based 3D arcade racing game with selectable tracks, AI rivals, nitro, a chase camera, finish timing, and a minimap.

## Run

```bash
npm install
npm start
```

Then open `http://localhost:4174`.

## Controls

- `W` or `ArrowUp`: accelerate
- `S` or `ArrowDown`: brake
- `A/D` or `ArrowLeft/ArrowRight`: steer
- `Shift`: nitro
- `R`: restart the current race
- `Esc`: track menu

Touch controls appear on smaller screens.
